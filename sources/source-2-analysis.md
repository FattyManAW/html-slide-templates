## 配色與設計趨勢（Page 1 觀察）

從首頁 90+ 模板樣本觀察：

- **主色調**：藍色系佔主導（B2B/科技），橙色/珊瑚色用於創意/時尚
- **佈局**：Hero image + 卡片網格 + CTA 為最常見模式
- **字體**：Sans-serif 主導（Inter/Roboto/Montserrat），少數使用 Playfair Display/Georgia
- **響應式**：全部使用 Bootstrap 或自適應 grid
- **動畫**：滾動觸發 fade-in 為主要模式

## 與 Source 1（beautiful-html-templates）的差異

| 維度 | beautiful-html-templates (S1) | nicepage.com (S2) |
|------|------|------|
| 設計目的 | 專為 coding agent 簡報設計 | 通用網站建置 |
| 模板粒度 | 完整 slide deck（8-20 頁） | 單頁/多頁網站 |
| 設計語言 | 高度風格化（34 種獨特美學） | 商業化、標準化 |
| coding agent 友好度 | ⭐⭐⭐ AGENTS.md + index.json | ⭐ 需手動解析 HTML |
| 元件可複用性 | 設計系統級（CSS variables + 裝飾詞彙） | Block 級（獨立 HTML section） |
| 簡報適用性 | 直接可用 | 需轉換（網站 → 簡報） |

## 素材庫使用建議

1. **簡報模板優先使用 S1** — 34 模板專為簡報設計，有完整工作流
2. **S2 用於補充**：
   - Landing page 風格的 Hero section → 簡報封面靈感
   - Block 設計元件 → 可複用組件庫（Testimonials、Team、Pricing）
   - 商業配色方案參考
3. **重點關注類別**：Business & Law、Technology、presentation keyword → 篩選出 50-100 個高品質簡報適用樣本
